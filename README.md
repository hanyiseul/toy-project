# 📂 Project List

## 📘 Vocabulary Book System

영어 단어와 뜻을 관리할 수 있는 CRUD 기반 웹 애플리케이션입니다.
Spring Boot와 JPA를 활용하여 백엔드를 구성하고,
모달 UI와 async/await를 적용해 사용자 경험을 개선했습니다.


### 주요 기능
- 단어 등록 / 수정 / 삭제
- 단어 목록 조회 (카드 UI)
- 단어 상세 조회 (모달)
- 키워드 검색 기능
- 페이징 처리 (Pageable)
- async/await 기반 비동기 모달 처리

**Frontend**
- Thymeleaf
- HTML
- CSS
- JavaScript

**Backend**
- Java 17
- Spring Boot

**Database**
- MySQL
